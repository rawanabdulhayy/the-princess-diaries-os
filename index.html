Export & Archive

Free-tier mitigation: export everything to a JSON file you keep yourself, then optionally clear old rows from the live database.
Export

Downloads a single JSON file with everything — log entries, inventory, planner templates and days. Keep these files somewhere safe; you can re-import any of them later to browse offline or restore into a fresh project.

Export & Clear a Date Range

Exports log entries in a date range, then — only after you confirm the download succeeded — deletes those rows from the live database to keep it small. Inventory and planner templates are never touched by this.

Import

Load a previously exported JSON file back in — either to browse it read-only in this tab, or to restore it into the database (only do this on a fresh/empty project to avoid duplicates).
